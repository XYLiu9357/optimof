# OptiMOF - RAC Extractor

Revised auto-correlated function (RAC) is a proven feature representation of nano-porous material that can be very effective in machine learning. The OptiMOF RAC extractor efficiently computes and represents the RAC-related features. Results will be stored in the temp directory created by [this script](src/model_features/feature_extraction.py) and **will be removed** once feature extraction is completed.
